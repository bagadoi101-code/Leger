Google Drive 設定說明
首次使用需設定 Client ID
?
Setup

第一次使用 Google Drive 同步，需要設定一個 Client ID（免費，5 分鐘）：

  1. 前往 Google Cloud Console，登入
  2. 建立新專案（隨便命名，例如「我的記帳」）
  3. 左側選單 → APIs & Services → Library，搜尋並啟用「Google Drive API」
  4. 左側選單 → APIs & Services → OAuth consent screen，選 External，填基本資訊（App 名稱、你的 email），Scopes 加入 drive.appdata，Test users 加自己的 Google 帳號
  5. 左側選單 → Credentials → Create Credentials → OAuth client ID → Web application
  6. Authorized JavaScript origins 加入：當前頁面的網址（例如 file:// 或你架設的網址）
  7. 複製 Client ID，貼到下方欄位
Client ID
提示：如果只在自己手機使用、不想設定 OAuth，建議改用「匯出 JSON 備份」定期手動備份到雲端硬碟，更簡單。
